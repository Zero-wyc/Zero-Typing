# Zero-Typing

#### 介绍

拼音打字玩具。
Chinese type test with PinYin.

#### 软件架构

Vite React generated.

-   React-Router
-   Redux

`master` 分支为构建后代码

#### 项目开发

1.  先切换至`dev`分支
2.  然后使用`pnpm`启动

```shell
pnpm install
pnpm dev
```

---

#### 本项目基于[typing-cn](https://github.com/barneyzhao/typing-cn)进行简单修改，并更换基础词库

#### 并添加 `.github/workflows/gh-pages.yml`进行GitHub Action实行自动网页构建
